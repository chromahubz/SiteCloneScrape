Project Information

Each project stores comprehensive information:

Basic Info

  • - Company name and website URL
  • - Industry and business category
  • - Date scraped and last modified
  • - Project status and priority

Scraped Data

  • - Complete website content
  • - Contact information and emails
  • - Images and assets
  • - Sitemap and page structure

Design Versions

  • - All AI-generated designs
  • - Model used for each version
  • - Screenshots and previews
  • - Download links for code

Outreach History

  • - Generated email templates
  • - Sent emails and responses
  • - Follow-up schedule
  • - Conversion status

Project Actions

Regenerate Design

Export Data

Share Projects

Duplicate Projects

Clone projects to create variations or use as templates for similar websites.
