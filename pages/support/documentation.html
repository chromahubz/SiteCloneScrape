Getting Started

Quick Start Guide

Welcome to SiteClone Pro! This guide will help you get started in minutes.

Prerequisites

  • - Valid AppSumo license code (or subscription)
  • - API keys for AI models (GPT-4, Claude, or Gemini)
  • - Modern web browser (Chrome, Firefox, Safari, or Edge)

Step 1: Account Setup

  1. Visit the SiteClone Pro dashboard
  2. Click "Get Started" or "Sign Up"
  3. Enter your AppSumo license code or create a subscription account
  4. Verify your email address
  5. Complete your profile setup

Step 2: Configure API Keys

To use AI recreation features, add your API keys:

  1. Navigate to Settings > API Configuration
  2. Add your OpenAI API key (for GPT-4)
  3. Add your Anthropic API key (for Claude)
  4. Add your Google API key (for Gemini)
  5. Click "Save" and test the connection

Note: You only need one API key to get started. We recommend starting with Gemini for cost-effectiveness.

Step 3: Your First Project

  1. Click "New Project" or "Scrape Website"
  2. Enter a target URL (e.g., https://example.com)
  3. Click "Start Scraping"
  4. Wait for the scraping process to complete (1-3 minutes)
  5. Review the extracted data
  6. Click "Generate Design" to create an AI-powered recreation

Success!

You've just created your first website recreation. Download the files, generate outreach emails, or save the project for later.

Website Scraping Guide

How Scraping Works

Our scraping engine uses Firecrawl to intelligently extract website content, structure, and assets.

Supported Content Types

Text Content
  • - Headings and body text
  • - Navigation menus
  • - Contact information
  • - Business details
Media Assets
  • - Images (JPG, PNG, WebP)
  • - Logos and icons
  • - Background images
  • - Alt text and captions

Best Practices

Use Full URLs

Always include https:// in your URLs

Start with Homepage

Scrape the homepage first for best sitemap discovery

Review Data Quality

Check the scraped data before generating designs

Limitations

  • - Maximum 100 pages per scrape session
  • - JavaScript-heavy sites may have reduced data extraction
  • - Login-protected content cannot be accessed
  • - Some sites may block scraping attempts

API Reference

REST API

Integrate SiteClone Pro into your own applications with our REST API.

Base URL

https://api.siteclonepro.com/v1

Authentication

// Include your API key in headers
Authorization: Bearer YOUR_API_KEY

Endpoints

POST /scrape

Scrape a website and extract data

View Example
curl -X POST https://api.siteclonepro.com/v1/scrape \
-H "Authorization: Bearer YOUR_API_KEY" \
-H "Content-Type: application/json" \
-d '{"url": "https://example.com", "max_pages": 50}'
POST /generate

Generate AI-powered website design

View Example
curl -X POST https://api.siteclonepro.com/v1/generate \
-H "Authorization: Bearer YOUR_API_KEY" \
-H "Content-Type: application/json" \
-d '{"project_id": "proj_123", "model": "gpt-4"}'
GET /projects

List all projects

GET /projects/:id

Get project details

Troubleshooting

Common Issues

Scraping Fails or Returns No Data

  • - Check if the URL is accessible in a browser
  • - Ensure you're using https:// not http://
  • - The site may be blocking automated access
  • - Try scraping a different page from the same site

AI Generation Takes Too Long

  • - GPT-4 can take 2-5 minutes for complex sites
  • - Try using Gemini for faster results
  • - Check your API key rate limits
  • - Reduce the amount of content being processed

API Key Errors

  • - Verify the API key is correct and active
  • - Check if you have available credits
  • - Ensure the key has proper permissions
  • - Try regenerating the key from the provider

Generated Design Not as Expected

  • - Try a different AI model
  • - Review the scraped data for quality
  • - Add more specific customization options
  • - Generate multiple variations and compare
